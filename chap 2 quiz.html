Section A: Multiple Choice (10 Questions)

1. Which method of data transmission allows data to flow in both directions simultaneously?





Explanation: Correct! Full-duplex allows data to flow simultaneously to and from the sender and receiver.

2. What is the role of the 'trailer' in a data packet?





Explanation: Correct! The trailer tells the receiving device it has reached the end of the packet.

3. Which transmission method sends data one bit at a time down a single channel?





Explanation: Correct! Serial data is transmitted sequentially down one channel.

4. What is 'packet switching'?





Explanation: Correct! Packets take different routes determined by the router.

5. Which hardware interface is now the standard for connecting devices?





Explanation: Correct! USB is the connection of choice for manufacturers.

6. What type of error detection adds a 1 or 0 to a 7-bit byte?





Explanation: Correct! Parity checks ensure the number of 1s is odd or even based on the protocol.

7. Which type of error detection sends a copy of the data back to the sender?





Explanation: Correct! In an echo check, the receiver immediately bounces the data back.

8. What is the primary purpose of a check digit on a barcode?





Explanation: Correct! Check digits spot human errors like mis-typing a numerical code.

9. Symmetric encryption uses how many keys?





Explanation: Correct! Symmetric uses one shared key to cipher and decipher.

10. The Caesar cipher is an historical example of what?





Explanation: Correct! It encrypts data by shifting the alphabet.
