1. What is the primary role of the Central Processing Unit (CPU)?





Explanation: Correct! The CPU's job is to process the instructions and data entered through input devices [cite: 783-784].

2. Which CPU register holds the address (location) of the next instruction to be carried out?





Explanation: Correct! The Program Counter contains the address of the next instruction to be fetched[cite: 809].

3. Which type of primary memory is volatile and loses its contents when the power is turned off?





Explanation: Correct! RAM is volatile memory that does not retain data without power [cite: 819-820].

4. In an automated system, which component turns electrical signals from the microprocessor into physical movements?





Explanation: Correct! Actuators are mechanical motors that carry out actions based on instructions[cite: 1038].

5. What defines an "embedded system"?





Explanation: Correct! Embedded systems perform highly specific tasks, like in a dishwasher or satellite navigation [cite: 874-877].

6. How does a Capacitive touch screen determine where a user has touched?





Explanation: Correct! Capacitive screens rely on the human body being naturally conductive [cite: 932-933].

7. Which type of secondary storage reads and writes data by burning 'pits' and leaving 'lands' on a surface using a laser?





Explanation: Correct! Optical drives like CDs, DVDs, and Blu-rays use lasers to read/write pits and lands[cite: 1144, 1156].

8. What happens when a computer utilizes "Virtual Memory"?





Explanation: Correct! When RAM is overloaded, the OS designates space on the hard disk to act as temporary memory [cite: 1166-1167].

9. Which address uniquely identifies a device's network interface controller (NIC) and is permanently set during manufacture?





Explanation: Correct! The Media Access Control (MAC) address is preset at manufacture and never changes [cite: 1187-1189].

10. Which device converts signals from a telephone or fibre optic line into data that is readable by a computer?





Explanation: Correct! A modem converts these signals, and is often built directly into modern routers [cite: 1222-1223].
